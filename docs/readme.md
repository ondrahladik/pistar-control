# Dokumentace

Tato složka shrnuje veřejné části aplikace a jejich účel.

## Přehled

- [REST API](api.md): endpointy pro stav, konfiguraci a přepínání profilů
- [Webové rozhraní](web.md): stránky `/`, `/config`, `/docs` a přihlášení
- [Telegram bot](telegram.md): dálkové ovládání přes dashboard message, tlačítka a aliasové příkazy

## Jak dokumentaci číst

Pokud chceš aplikaci ovládat programově, začni v [api.md](api.md).
Pokud řešíš práci v UI, pokračuj na [web.md](web.md).
Pro nastavení bota a ovládání z Telegramu je určené [telegram.md](telegram.md).
